NodeJS : Mise à jour Node/NPM impossible Windows.

Je souhaite mettre à jour Node mais même après une réinstallation du package d'installation ; "node -v" ne retourne pas la version espérée.

Il semblerait que plusieurs version de Node sont installées sur la machine (version par défaut de Visual Studio).

Il faut désinstaller Node et supprimer toutes ses références.

Supprimer les références à Node (source)

  • Supprimer Node des "Programmes et fonctionnalités"
  • Vérifier que le dossier suivant est bien supprimer => C:\Program Files (x86)\Nodejs
  • Vérifier que le dossier suivant est bien supprimer => C:\Program Files\Nodejs
  • Supprimer => C:\Users\{User}\AppData\Roaming\npm (ou %appdata%\npm)
  • Supprimer => C:\Users\{User}\AppData\Roaming\npm-cache (ou %appdata%\npm-cache)
* Vérifier également sur les autres disques durs.

Clean Install / Update de NodeJs

Node nous recommande pour chaque "mise à jour" de faire une installation via le package d'install et non via ligne de commandes (version actuelle). Donc il faut télécharger le dernier ".msi" correspondant à votre type de processeur/OS.
