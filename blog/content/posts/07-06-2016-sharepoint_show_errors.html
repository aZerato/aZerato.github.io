Sharepoint 2007 : show errors

Vous voulez afficher autre chose que : 'Une erreur est survenue', alors faites ce qui suit dans le web.config de la webApp SharePoint.

Vous voulez afficher autre chose que : 'Une erreur est survenue', alors faites ce qui suit dans le web.config de la webApp SharePoint.

Custom Error

Activer les "custom errors" en vous rendant dans le tag : <system.web> ... </system.web>

Trouver le tag : <customErrors mode="On" /> à changer en <customErrors mode="Off" />

Stack Trace

Activer la "stack trace" en vous rendant dans le tag : <SharePoint> ... </SharePoint>

Trouver le tag : <SafeMode CallStack="false" ... /> à changer en <SafeMode CallStack="true" .../>

Debug Mode

Trouver le tag : <compilation batch="false" debug="false" /> à changer en <compilation batch="true" debug="true"/>

Ajouter une Stack Trace ASP.NET

Il faut vous rendre au niveau du tag : <system.web> ... </system.web>

Et ajouter le tag : <trace enabled="true" pageOutput="true"/>
