Sharepoint - erreur "SqlSessionState"

Vous pouvez obtenir cette erreur si vous tentez d'ouvrir un site.

Travaillant sur une solution en multi projets sur du Sharepoint 2013, au moment du build sur le projet principal, on obtenait une telle erreur. Ce projet était pourtant bien référencé dans le main project.

Se rendre sur ce fameux 'main project', se rendre dans 'package > package.package'. L'ouvrir et se rendre dans l'onglet 'Advanced > Add > Add assembly from project' et ajouter le projet correspondant à la DLL manquante.

Sharepoint 2013 - Internal DLL project : "File not found"

I've worked on Sharepoint 2013 multi projects solution. At build time, i've got an error 'file not found' about another project in the same solution, but that was referenced.

I've worked on Sharepoint 2013 multi projects solution. At build time, i've got an error 'file not found' about another project in the same solution, but that was referenced.

Go to the main project, open 'package > package.package' file. Find the 'Advanced' tab, go to 'Add > Add assembly from project'. Add missing project.
