Sharepoint 2007 - Windows server 2003 64 bits : problème à l'installation

Message d'erreur 'This product requires ASP.NET v2.0 ...' alors que ce dernier est bien installé.

Se rendre dans 'ISS Manager', dans la partie gauche vous trouvez un dossier 'Web Services Extensions', en cliquant dessus la partie droite change, trouvez le ou les extensions concernant 'ASP.NET V2.0' et 'autorisée' les (allow).

Sharepoint 2007 - Windows server 2003 64 bits : install problem

Error message 'This product requires ASP.NET v2.0 ...' but, it's already installed.

Go to the 'ISS Manager', in the left part you can find a 'Web Services Extensions' folder. When you click on it, the right part changed. You need to 'allow' the different(s) extension(s) 'ASP.NET V2.0'.
