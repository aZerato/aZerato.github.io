Maintenant que vous savez naviguer vers le mode debug et surtout vers le fichier de configuration "launch.json", vous pouvez éditer la manière de lancer votre programme.

Dans un précédent article j'avais présenté différentes lib node dont "commander" et "grunt-execute". Dans le code de ce site j'utilise donc une tache pour "compiler" / "indexer" mes articles.

Voici a quoi ressemble mon "launch.json" pour executer uniquement l'action "indexing" de mon tool en cli : { "type": "node", "request": "launch", "name": "Lancer le programme", "program": "${workspaceRoot}/cli/program.js", "cwd": "${workspaceRoot}", "args": [ "indexing", "-d", "-p", "blog/content/posts/" ] }, Le fichier en question se trouve sur mon Github : lien.
