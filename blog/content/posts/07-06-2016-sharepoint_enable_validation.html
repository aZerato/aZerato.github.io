Sharepoint 2007 : 'Enable Validation' error

Attention quand vous créez un element via C# tel un DropDownList pour créer en HTML un select, il y a un binding qui est fait. Par conséquent, si vous effectuez des modifications via du js sur le contenu du select vous risquez bien de rencontrer une telle erreur.

Si vous avez des elements html que vous risquez de modifier via js utiliser plutôt l'objet HtmlGenericControl pour le créer.
