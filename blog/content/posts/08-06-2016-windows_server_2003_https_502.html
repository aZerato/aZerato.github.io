Windows Server 2003 : HTTPS 502

Vous effectuez des requêtes HTTPS mais nous obtenions directement un code d'erreur 502.

Il était nécessaire de requéter un WebService SSO côté serveur (pour limiter la plage IP à authoriser) par code C#. Seulement le code fonctionnant très bien sur une machine avec Windows 8 nous retournait une erreur "System.IO.IOException: Received an unexpected EOF or 0 bytes from the transport stream" sur le serveur Windows 2003.

Il fut nécessaire de faire une mise à jour permettant de revoir le système d'encryption pour le requêtage HTTPS.
