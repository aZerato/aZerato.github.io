When I edit my HTML code with Visual Studio Code some spaces are deleted when i save. My HTML code is automatically formatted that break "cut & past" text structure.

When I write a new post with "code samples", I use a custom angular component. "<prism-component>" is based on Prism.js library, and create automatically "<pre><code>" tags.

If I save my file with "<code>" tag, the inner text is not formatted, but if I use my "<prism-component>" that inner text is formatted.

To deactivate this feature : "CTRL + SHIFT + P" / "CMD + SHIFT + P". After that, type "formatter" and select "Formatter Config", in formatter.json file change "onSave" value to false.
