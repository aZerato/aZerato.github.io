Sharepoint 2013 : "Unable to import WebPart"

Solution SP 2013 découpée en plusieurs sous projets. Le projet X référence bien le projet Y, il faut se rendre dans le projet principal de votre solution Sharepoint, se rendre dans 'package' > 'package.package', et se rendre dans l'onglet 'Advanced' > 'Add' > 'Add assembly from project' et ajouter le projet qui manque.

Sharepoint 2013 : "Unable to import WebPart"

SP 2013 solution, with some under projects. Even if the project Y is referenced in project X, go to the main project Sharepoint, open folder 'package' > 'package.package'. Then, open the 'Advanced' tab > 'Add' > 'Add assembly from project'.
