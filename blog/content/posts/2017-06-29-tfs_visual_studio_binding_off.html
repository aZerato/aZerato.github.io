J'ai rencontré un problème il y a peu sur une solution versionnée sous TFS. J'étais sur Visual Studio 2010.

Mes modifications n'étaient pas prises en compte. Il était donc impossible de pusher mes corrections sur le serveur. Refaire le mapping avec un workspace ne m'aider pas.

Il faut donc réinitialiser le binding des projets de votre solution.

Pour ce faire rendez-vous dans le menu "File > Source Control > Change Source Control". Vous verrez une fenêtre avec l'ensemble de votre solution et sans aucun doute la colonne "Server Name" incorrecte.

Vous séléctionnez les projets posant problème et vous cliquez sur "bind".

I've gotten a problem with a solution on Visual Studio 2010.

TFS was unable to detect any modification.

We need to reactivate binding.

Go to "File > Source Control > Change Source Control". Now, you can see a new window showing all your solution mapping. Some values can be incorrect in "Server Name" column.

Select all your incorrect projects and click on "bind" button.
