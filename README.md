# aZerato.github.io

My github.io website

## How to start

If not already done :

- Download node.js
- Install bower globally
> npm install bower -g
- Install grunt globally
> npm install grunt-cli -g

## Init

> npm install

> bower install

## Grunt Tasks 

Run local server & open default browser
> grunt
or
> grunt default

Run jshint
> grunt check

Concat/minify app files in '/build'
> grunt build
